* Version
Open CMD, write: java -version

* Run a java file

1. Compile a java file
Open CMD, go to directory of the file, write:

javac <file.java> // for file
javac <package name>/<file.java> // for file with package
	e.g.:
	javac Main.java
	javac packagea/Main.java

2. Run a java file

java <file.java> // for file
	e.g.:
	java Main

java <package name>/<file.java> // for file with package

note: If it doesn't work, then comment out the .java package.
